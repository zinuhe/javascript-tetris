# javascript-tetris

## Create

```shell
npm create vite@latest
```

## Install dependencies

```shell
pnpm install
```

## Instal linter

```shell
pnpm install standard -D
```

## Config linter

[Documentation](https://eslint.org/docs/latest/use/configure/)

On **`package.json`** add at the end

```javascript
"eslintConfig": {
    "extends":[
        "standard"
    ]
  }
```

## Canvas

Add a canvas to draw on it

# To Know

# To RUN

```shell
pnpm run dev
```
